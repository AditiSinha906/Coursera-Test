step 1:

use md to make a new folder in the repository clone

step 2:

make a html file and save it in the new folder with .html and write your code.. save it!

Step3:

check the status"git status" if there is a new file it will show "./"

step 4:

use "git add ." to include it in the folder

Step5:

exit the folder using "cd..", check the status "git status"

step6:

commit the change by "git commit -m "your msg""

step 7:

get the status "git status"

Pull first "git pull"

push later "git push"

last step:

check the status again, refresh your browser and then do the execution process.
